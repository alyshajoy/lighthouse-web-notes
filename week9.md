# AJAX

### AJAX
* asynchronous JS and XML
* invented by Microsoft
* XML stands for eXtensible Markup Language

```xml
<user>
  <username>alice</username>
  <password>1234</password>
</user>
```

```json
{
  "username": "alice",
  "password": "1234"
}```

* JS is going to make our HTTP requests
* XMLHTTPRequest object (XHR)
